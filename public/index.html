K-Pay File Uploader

Upload Files To K-Pay Easily!

Simply fill out your company shortname, api key, username and password and select the files to upload. The username and password must belong to an employee in the company, not a super user that has access to the company. Additionally, the user must have access to the employee(s) you are uploading files to and must also have access to document storage. Multiple files can be selected from the file upload selected by holding command/control while selecting files.

  1. Download the sample mapping csv file.
  2. Put the name of the file that this line corresponds to in the first column.
  3. Put the System ID of the record you wish to attach the file to in the "system_id" column. You can find this by adding the System ID or Account ID fields to your record list in K-Pay.
  4. (Optional) Include a "Document Type" in the document information in K-Pay by supplying a Document Type Name in the "document_type_name" column. Note, the name must match exactly to the name in K-Pay.
  5. (Optional) Include a description that will be included in the document description in K-Pay.
  6. (Optional) If your file is a photo that you wish to put in the employee photo field put a "Yes" in this column. Note that if you put a "Yes" in this column the file will not be stored in the Document Storage area of the employee but will only be visible in the photo area of the Account Demographics section. As such the Document Type and Description columns will be ignored. Also note, I am not validating or manipulating any files here so please make sure your file is a valid image and is formatted correctly before uploading.
