K-Pay File Uploader

Easily upload documents to K-Pay!

Instructions:

  1. Using the form below, fill out your Company Shortname and API Key associated with your K-Pay account.
  2. Fill in a Username and Password with which the application will complete the upload.
    3. The username and password must belong to an employee in the company, not a "super user" that has access to the company. The user must have access to the employee(s) you are uploading files to and must also have access to document storage.
  3. Upload a CSV Mapping File (see detailed instructions below, use the example CSV file as a guide).
    4. The CSV Mapping File will tell the system where to place the files in K-Pay and will need the following columns: file_name, system_id, document_type_name, description, and employee_photo.
    Column Name Required? Description
    file_name YES Indicate the name of the file that this line corresponds to (There should be a line for each file uploaded).
    Note, the file extensions are not required. (Example, can either upload "resume" or "resume.pdf")
    The names of the files that you are uploading must be unique.

    Tip: When copying many files, it will save time to copy the files from the folder on your coputer into the csv mapping file. On a MAC, highlight all files and copy and paste the file names from the file manager into the csv mapping file. On a PC, highlight all files and copy the "file path" of the file names from the file explorer. Then, copy into your csv mapping file and manipulate in Excel (find and replace) so that just the file name is listed.
    system_id YES Indicate the System ID you wish to attach the file. The specific System ID will depend on the type of document you are uploading.
    For Employee Documents, this is the "Account ID" of the employee record. This can be found by adding the Account ID field to your record list in K-Pay.
    document_type_name NO Indicate the K-Pay Document type that matches exactly to the document type name in K-Pay.
    description NO Indicate the Description that will be included in the document description in K-Pay.
    employee_photo NO Indicate if file should be uploaded to employee photo area in Account Demographics (Must be in a.jpeg, .jpg, or .gif file format.) by putting a "Yes" in this column, otherwise leave blank.
    The Document Type and Description columns will be ignored.
    The file will not be stored in the Document Storage area, it will be uploaded to the photo area of the Account Demographics.
    There is no manipulation or validation of the file, so ensure the file is a valid image file type and is formatted correctly before uploading.


  4. Select the File(s) to Upload to K-Pay.
    Multiple files can be selected from the file upload selected by holding command/control while selecting files. All files selected here need to also be included in the mapping CSV file.
  5. Click the Submit button.
  6. If there are errors, you can download the Errors CSV File by clicking Download Errors.
